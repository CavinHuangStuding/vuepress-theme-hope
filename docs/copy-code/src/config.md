## baseLang

- Type: `string`
- Default: `'en-US'`

The language of the home directory.

### selector

- Type: `string | string[]`
- Default: `'.theme-default-content div[class*="language-"] pre'`

Code block selector

### duration

- Type: `number`
- Default: `300`

Prompt message display time

### showInMobile

- Type: `boolean`
- Default: `false`

Whether to display on the mobile side
