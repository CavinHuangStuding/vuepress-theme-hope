## baseLang

- Type: `string`
- Default: `'en-US'`

The language of the home directory.

### selector

- Type: `string | string[]`
- Default: `'.theme-default-content div[class*="language-"] pre'`

Code block selector

### duration

- Type: `number`
- Default: `2000`

Hint display time, setting it to `0` will disable the hint.

### showInMobile

- Type: `boolean`
- Default: `false`

Whether to display copy button on the mobile device
