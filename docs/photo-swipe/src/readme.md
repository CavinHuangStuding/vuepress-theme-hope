Let your images support preview, zoom, share, swipe view and download.

## How to use

### Install

<CodeGroup>
<CodeGroupItem title="yarn">
```bash
yarn add -D vuepress-plugin-photo-swipe
```
</CodeGroupItem>

<CodeGroupItem title="npm">
```bash
npm i -D vuepress-plugin-photo-swipe
```
</CodeGroupItem>
</CodeGroup>

### Usage

```js
// .vuepress/config.js
module.exports = {
  plugins: ["photo-swipe"],
};
```
